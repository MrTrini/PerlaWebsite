# Perla Lutchman — Website

Two static sites served via GitHub Pages:

- **index.html** — landing page linking both sites
- **branding.html** — Brand Strategy & Consulting
- **real-estate.html** — Luxury Real Estate (Miami)

Built as plain HTML/CSS/JS (no build step). Optimized for SEO/ASEO (structured data, meta, sitemap, llms.txt), mobile + desktop responsive, and accessibility.
